# Vaishnavi Murthy — Art Portfolio

A self-contained static site. Open `index.html` locally, or host with GitHub Pages.

- `index.html` — home (journey + "where I am now")
- `works.html` — full gallery (filterable, lightbox)
- `styles.css`, `app.js`, `data.js` — styling, behaviour, gallery data
- `images/` — artworks
